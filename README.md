# STEPS TO RUN HelloWorld.java PROGRAM ON CMD

1. Installed Java JDK and checked installation using java -version and javac -version.
2. Opened a text editor such as Notepad. Written the code there and Saved the file as HelloWorld.java.
3. Opened Command Prompt (CMD). Navigated to the folder where HelloWorld.java is saved using the cd command.
4. Compiled the Java program using javac HelloWorld.java.
5. Ran the program using java HelloWorld.
6. The program executed and displayed the output in CMD.
